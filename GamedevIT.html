Лекция: C# и C++ в игровой разработке

Язык программирования C#

C# - современный объектно-ориентированный язык программирования, разработанный компанией Microsoft. Он широко используется в игровой разработке, особенно при создании игр на платформе Unity.

Особенности C#:

  • Объектно-ориентированный: C# поддерживает принципы ООП, такие как наследование, инкапсуляция и полиморфизм.
  • Управляемый код: Код на C# компилируется в промежуточный язык (CIL), который исполняется виртуальной машиной Common Language Runtime (CLR).
  • Богатая стандартная библиотека: C# поставляется с обширной стандартной библиотекой, включающей в себя множество полезных классов и методов.

Язык программирования C++

C++ - мощный и эффективный язык программирования, широко используемый в игровой индустрии благодаря своей производительности и возможностям низкоуровневого управления ресурсами.

Особенности C++:

  • Высокая производительность: C++ обеспечивает близкое к металлу управление памятью и высокую скорость выполнения.
  • Низкоуровневое программирование: C++ позволяет программисту полностью контролировать использование памяти и процессора.
  • Мультипарадигменный: C++ поддерживает как процедурное, так и объектно-ориентированное программирование.

Игровые движки Unity и Godot

Unity и Godot - это два популярных игровых движка, которые позволяют разработчикам создавать высококачественные игры для различных платформ.

Unity

Unity - это мощный и гибкий игровой движок с широким набором функций и инструментов для создания игр 2D и 3D. Он поддерживает множество платформ, включая iOS, Android, Windows, macOS и многие другие.

Godot

Godot - это бесплатный и открытый исходный код игровой движок, который предоставляет удобный в использовании интерфейс и множество инструментов для разработки игр различных жанров.

Приветствие Напишите программу на C++, которая запрашивает у пользователя его имя и выводит на экран сообщение "Привет, [имя]!".

Расчет площади прямоугольника Напишите программу, которая запрашивает у пользователя длину и ширину прямоугольника, а затем выводит его площадь.

Определение четности числа Напишите программу, которая запрашивает у пользователя число и выводит сообщение "Число четное" или "Число нечетное", в зависимости от того, четное оно или нет.

Вот примеры решений этих заданий на C# и C++: 

    
        using System;

        class Program
        {
            static void Main(string[] args)
            {
                // 1. Приветствие
                Console.Write("Введите ваше имя: ");
                string name = Console.ReadLine();
                Console.WriteLine($"Привет, {name}!");
        
                // 2. Расчет площади прямоугольника
                Console.Write("Введите длину прямоугольника: ");
                double length = double.Parse(Console.ReadLine());
                Console.Write("Введите ширину прямоугольника: ");
                double width = double.Parse(Console.ReadLine());
                double area = length * width;
                Console.WriteLine($"Площадь прямоугольника: {area}");
        
                // 3. Определение четности числа
                Console.Write("Введите число: ");
                int number = int.Parse(Console.ReadLine());
                if (number % 2 == 0)
                    Console.WriteLine("Число четное");
                else
                    Console.WriteLine("Число нечетное");
            }
        }
