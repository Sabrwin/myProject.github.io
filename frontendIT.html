Лекция: Введение в HTML, CSS и JavaScript

Добро пожаловать на лекцию о трех основных технологиях веб-разработки: HTML, CSS и JavaScript. HTML (HyperText Markup Language) HTML является основой веб-страниц. Это язык разметки, который определяет структуру контента на веб-странице. Основные концепции HTML: HTML использует теги для определения различных элементов на странице. Вот несколько основных тегов:



    
        <html>: Определяет начало и конец HTML документа. <br>
        <head>: Содержит метаданные документа, такие как заголовок страницы и ссылки на внешние ресурсы. <br>
        <body>: Содержит основное содержимое страницы, такое как текст, изображения, таблицы и другие элементы. <br>
        <h1>, <h2>, <h3>, ... <h6>: Определяют заголовки разных уровней. <br>
        <p>: Определяет абзац текста. <br>
        <img>: Вставляет изображение. <br>
        <a>: Создает ссылку. <br>
        <ul>: неупорядоченный список. <br>
        <ol>: нумерованнный список. <br>

CSS (Cascading Style Sheets) CSS используется для стилизации веб-страниц. Он определяет внешний вид и оформление элементов на странице. Основные концепции CSS: CSS использует правила, состоящие из селекторов и объявлений стилей. Селекторы определяют, к каким элементам применяются стили. Объявления стилей определяют, как выглядят выбранные элементы.

JavaScript делает ваши веб-страницы интерактивными. Это язык программирования, который добавляет динамическое поведение к HTML-элементам. Основные концепции JavaScript: JavaScript позволяет выполнять следующие действия: Изменять содержимое HTML. Изменять стили CSS. Работать с событиями, такими как клики и наведения мыши.

HTML:
Создание веб-страницы:
